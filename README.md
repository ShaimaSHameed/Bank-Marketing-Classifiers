# Bank Marketing Classifiers
**Machine Learning Course Project**

Compared three supervised classifiers predicting whether a bank client subscribes to a term deposit. Dataset of 45,211 records across 16 features.

## Results
| Model | Accuracy | AUC |
|-------|----------|-----|
| Logistic Regression | 84.07% | 0.9147 |
| Decision Tree | 84.45% | 0.8938 |
| Naive Bayes | 72.50% | 0.8163 |

## Stack
Python · scikit-learn · pandas · matplotlib
